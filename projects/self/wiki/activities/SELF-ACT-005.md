# AI第三者指摘→偽の確証（interest≠intent）による確信度上げの制止を自己ドッグフーディングで観察

対象仮説: [[SELF-H-015]]

## テストカード（検証前に記入・後から書き換えない）

- **目的**: [[SELF-H-015]] の効き所「**AIの第三者指摘 → 実践者が偽の確証（interest≠intent）による確信度の上げを止める**」が実際に起きるかを、自己ドッグフーディングで観察する。
  - **主に反証条件①**（AIが指摘しても行動が変わらないか）を測る。
  - **反証条件②**（自制・チェックリストより第三者性が優位か）は、被験者＝実践者本人の n=1 自己観察では**弱くしか測れない**（「AIが気づかせた」のか「自分でも気づけた」のかを完全には切り分けられない）ことを前提として明記する。外部被験者での検証は将来のスコープ。

- **方法**: demo（自己ドッグフーディング観察。外部被験者を伴わないため、インタビュースクリプトは作らず下の記録シートで代替する — 実例 [[SELF-ACT-003]] と同型）。実際のスキル運用（`/refine` `/plan` `/ingest` `/decide`）の中で、AIが「**好意的反応（interest）を確証（intent）と取り違えて確信度を上げようとした場面**」を第三者として指摘したイベントを、下の記録シートに1件ずつ記録し、その**直後の実践者の行動**を追う。各件で「AI指摘の前に実践者が自力で取り違えに気づいていたか」も正直に○×で記録し、第三者性の寄与を切り分ける。

  ### 記録シート（イベントごとに1行）

  | # | 場面（どのスキル・何の確信度/判断か） | AIの指摘内容（interest≠intent） | 指摘前に自力で気づいていたか（○=気づけていた/×=気づけていなかった） | 直後の行動（保留/据え置き・反証追加・根拠リンク補完・変化なし） | 進路変更したか（○/×） |
  |---|---|---|---|---|---|
  | 1 | | | | | |
  | 2 | | | | | |

- **指標**:
  1. AIが interest≠intent 型の指摘を出した回数（＝指摘機会）
  2. そのうち実践者が実際に**進路を変えた**回数（確信度の上げを保留/据え置き・反証を狙う検証を追加・根拠リンクを補完）
  3. そのうち「**指摘前は自力で気づいていなかった**」件数（＝第三者性が寄与した件数）

- **成功基準**（開始後は書き換えない）:
  - (a) interest≠intent の指摘機会が観測され、その**過半で進路変更**が起きる（＝反証条件①をクリア）。
  - (b) そのうち**少なくとも1件は「自力では気づけていなかった」事例**がある（＝反証条件②に部分的に触れる、第三者性の寄与を示す）。
  - (a)(b) を両方満たせば支持寄り、(a) のみなら判断保留、(a) を満たさなければ反証。
  - **弱い証拠として扱う**: 自己観察・n=1・自己奉仕バイアスの余地があるため、`sources/` とビューに「**実在被験者未検証**」を明示し、支持でも確信度を上げすぎない（成功基準未達なら原則上げない＝ SI-008 の3値判定に従う）。
